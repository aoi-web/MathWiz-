# MathWiz 🧠

A simple interactive math quiz web app built using HTML, CSS, and JavaScript.

## 🚀 Features

* Multiple choice questions
* Dynamic question rendering
* Score tracking
* Result screen after quiz
* Restart functionality

## 🛠️ Tech Used

* HTML
* CSS
* JavaScript (Vanilla)

## 🎮 How to Use

1. Select an answer
2. Click Submit
3. Complete all questions
4. View your final score
5. Restart and try again


## 📌 Future Improvements

* Add timer
* Random questions
* Difficulty levels
* Better animations
